# Netlify deploy checklist (fix 404)

If the build succeeds but the site shows **Page not found**:

1. **Publish directory must be empty**
   - Netlify → **Site configuration** → **Build & deploy** → **Build settings** → **Publish directory**
   - Leave it **blank**. Do not set `out`, `dist`, or `.next`.
   - The `@netlify/plugin-nextjs` plugin sets the correct output; a custom publish dir overrides it and causes 404.

2. **Build command**
   - Should be `bun run build` (or leave blank to use `netlify.toml`).

3. **Redeploy**
   - After changing Publish directory, run **Trigger deploy** → **Deploy site**.
